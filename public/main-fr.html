Créez vos propres jeux !

GDevelop est un logiciel de création de jeux open source, multiplateforme conçu pour être utilisé par tout le monde - aucune programmation n'est requise.

Evenements

Ce qui rend GDevelop unique et facile à utiliser sont les événements. Les événements sont un moyen puissant d'exprimer la logique de votre jeu, sans avoir à apprendre un langage de programmation.

Lorsque la touche espace est appuyée, l'animation du personnage et un son sont joués. Si une bombe touche le persommage, ils sont tous deux détruits.

Jeux HTML5 & natifs

L'architecture flexible de GDevelop vous permet de choisir si vous souhaitez créer un jeu HTML5 qui peut être joué sur le web et exporté vers Android, ou un jeu natif pour votre système d'exploitation. Ou activez les deux plates-formes et exportez votre jeu partout !

Télécharger GDevelop

Vous serez bientôt entrain de créer vos propres jeux, sans programmation !

  • Windows
  • Ubuntu
  • Autres linux
  • Mac OS X
  • iOS/Android
  • Sources

Téléchargez et installez GDevelop pour Windows :

Télécharger pour Windows

aussi disponible sans installateur

Ouvrez un terminal en appuyant sur Ctrl + Alt + T.

Puis tapez (et appuyez sur Entrer après chaque ligne) :

sudo add-apt-repository ppa:florian-rival/gdevelop
sudo apt-get update
sudo apt-get install gdevelop

Vous avez terminé ! Tapez GDevelop dans la barre de recherche d'Ubuntu pour commencer.

Vous pouvez obtenir GDevelop pour :

Fedora, Arch, OpenSuse
Arch (à partir des sources)

Il s'agit d'un paquet non officiel

N'oubliez pas que GDevelop est principalement testé sur Ubuntu. Vos commentaires sont les bienvenus sur le forum.
